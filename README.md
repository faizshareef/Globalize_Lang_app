Globalize is a lightweight Streamlit-based web application that breaks language barriers. It doesn't just translate text; it allows you to **visualize** the core themes through word clouds and **hear** the translation using high-quality Text-to-Speech (TTS).

## âœ¨ Features
- **Auto-Language Detection:** Identify the source language of any paragraph instantly.
- **Universal Translation:** Convert text into English or multiple target languages simultaneously.
- **Thematic Word Clouds:** Generates a visual summary of the most frequent English terms in your text.
- **Multi-Voice Playback:** Listen to your translations in native-sounding voices via Google TTS.
